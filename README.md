
# Ops Tracker

## What it is
An internal tool to track operational metrics, tasks, and processes for startups.

## Problem it solves
Startups need visibility into operations without bloated tools or dashboards.

## Tech stack
- Backend: Node.js + Express
- Frontend: React (or your choice)
- Database: TBD (Postgres, SQLite, or MongoDB)
- Dev: Git, VSCode, Postman
